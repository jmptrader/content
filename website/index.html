Introduction

The Project

PubSubSQL makes it simple, easy and quick to develop applications that require database and notification services. Publish-Subscribe messaging is integrated with the in-memory database as a single product. When the underlying data changes, client subscribers receive updates in soft real-time without having to poll the database or implement database trigger hooks.

PubSubSQL is written in Go and runs on Linux and Windows, with MAC OS X in the works. We provide unified APIs for Go, Java and .Net and will be adding Objective-C, Python, JavaScript and Ruby. PubSubSQL implements its own trimmed down flavor of SQL and has a dynamic table schema with columns created on demand.

We are happy to present you with our first release and hope that you will find the product fun to use.
